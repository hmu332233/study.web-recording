# study.web-recording

1. 카메라로 부터 입력 받기
  - `MediaDevie.getUserMedia`
  - https://developer.mozilla.org/ko/docs/Web/API/Media_Streams_API
2. 카메라로 받은 입력을 HTMLVideoElement를 통해서 실시간으로 출력
3. 카메라로 부터 받은 입력을 녹화
4. 녹화된 영상 미리보기
5. 녹화된 영상 다운로드
